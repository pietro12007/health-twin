# Contributing to My Future Health

**NEVER force push to main. Ever. If you need to undo something, ask the team first.**

## Workflow
- Always run `git pull` before starting work
- Never commit directly to main — create a branch
- Write clear commit messages describing exactly what changed
- Tell the team in the group chat what you pushed before closing your laptop
- Each team member uses their own `.env.local` with their own API key — never commit this file

## Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env.local` with your own `ANTHROPIC_API_KEY`
4. Run `npm run dev`
5. Open http://localhost:3000
